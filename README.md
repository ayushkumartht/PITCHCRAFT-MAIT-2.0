# PitchCraft 2.0 MAIT 🚀

PitchCraft 2.0 is a modern, high-performance web application designed for entrepreneurs and innovators to refine their pitches using AI-driven insights. Built with React, Vite, and Framer Motion, it offers a seamless and interactive experience.

## ✨ Features

- **Dynamic Landing Page**: A visually stunning introduction to the platform with smooth animations.
- **Interactive Dashboard**: Manage your projects and track your progress in real-time.
- **AI Scan**: Leverage advanced AI models to analyze and improve your pitch deck and presentation logic.
- **Premium Shop**: Access exclusive templates, resources, and tools to elevate your pitching game.
- **Customizable Settings**: Personalize your experience and manage your account preferences.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)
- **Styling**: Vanilla CSS with modern practices.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/YOUR_USERNAME/pitchcraft2.0-mait.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```

## 📂 Project Structure

- `src/pages`: Contains the main views (Landing, Dashboard, AIScan, Shop, Settings).
- `src/components`: Reusable UI components.
- `src/assets`: Static assets like images and icons.
